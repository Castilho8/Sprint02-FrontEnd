# CareFit+

Aplicação web de bem-estar e saúde corporativa com sistema de missões diárias, gamificação e loja de recompensas.

## Páginas

- **Dashboard** — visão geral do dia, progresso de vitalidade e missões em destaque
- **Missões** — lista de missões físicas e mentais com progresso e filtros por categoria
- **Loja** — resgate de recompensas usando pontos acumulados
- **Perfil** — dados pessoais, conquistas, integrações e configurações de privacidade (LGPD)

## Tecnologias

- HTML5, CSS3 e JavaScript
- [Bootstrap 5.3](https://getbootstrap.com/)
- [Bootstrap Icons 1.13](https://icons.getbootstrap.com/)

## Como executar

Abra o arquivo `index.html` diretamente no navegador. Não requer servidor ou instalação.
